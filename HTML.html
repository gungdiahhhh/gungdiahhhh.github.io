HTML

I. PENGERTIAN

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan merancang halaman web. Dalam dunia web development, HTML berfungsi sebagai kerangka dasar yang mendefinisikan struktur dan isi dari sebuah halaman web.

IX. TABEL

➥ Menggunakan tag table (memulai tabel), tr (membuat tabel), th (head tabel), td (isi tabel), caption (judul tabel), colgroup dan col (memberikan style tertentu pada kolom), thead (kepala), tbody (isi) dan tfoot (kaki)

Atribut :

  • border : menambahkan garis
  • cellspacing : jarak antar tabel
  • cellpadding : jarak teks dalam tabel
  • colspan : digunakan di tag <.td> untuk menggabungkan kolom (kanan-kiri)
  • rowspan : digunakan di tag <.td> untuk menggabungkan baris (atas-bawah)

Atribut Rules :

  • cols : border hanya dikolom
  • rows : border hanya digaris
  • all : border dikolom dan dibaris (cukup tambahkan atribut border)
  • none : tanpa border (tidak perlu tambahkan atribut border)

X. FORM

➥ Menggunakan tag form (memulai), input (memasukan atribut), button

Tag Input (atribut type)

  • type="text" : memasukan tulisan
  • type="password" : memasukan password (terlihat hanya titik-titik)
  • type="date" : memasukan tanggal
  • type="time" : menambahkan waktu/jam
  • type="datetime-local" : menambahkan tanggal dan jam
  • type="month" : menambahkan bulan dan tahun
  • type="week" : menambahkan minggu
  • type="radio" : memasukan pilihan (tambahkan atribut name agar hanya 1 yang dapat dipilih dan value untuk diproses didatabase)
  • type="checkbox" : menambahkan checklist
  • type="color" : memberikan pilihan warna
  • type="email" : menambahkan email
  • type="url" : menambahkan url
  • type="search" : menambahkan fitur pencarian
  • type="number" : menambahkan angka
  • type="tel" : menambahkan nomor telp (dimobile keyboard angka otomatis muncul)
  • type="file" : menambahkan image, video, audio
